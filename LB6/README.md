Описание

В лабораторной работе реализованы два способа построения бинарного дерева:
1. Рекурсивный (`build_tree_recursive`)
2. Итеративный (`build_tree_iterative`)

Дерево представлено в виде словаря.  
В корне дерева находится число, задаваемое пользователем, высота дерева также задается пользователем.  

Левый и правый потомки вычисляются по формулам варианта 9:  
- Left = root * 2 + 1  
- Right = 2 * root - 1  


Результаты
- Реализован бенчмарк времени работы обеих функций с помощью модуля `timeit`.
- Построен график зависимости времени построения дерева от высоты.
- Сделаны выводы о сравнении двух подходов.
